# 🚀 My Portfolio - Nanang Sanjaya

Website portfolio **Meta Ads Specialist** — Nanang Sanjaya.

## 📦 Cara Menjalankan di Local

```bash
git clone https://github.com/nanangsanjaya0896/My-Portfolio.git
cd My-Portfolio
npm install
npm run dev
```

Website berjalan di: **http://localhost:3000**

## 📁 Struktur Folder

```
My-Portfolio/
├── index.html        ← Halaman utama
├── styles.css         ← Custom styles
├── script.js          ← JavaScript interaksi
├── ai/
│   └── chatbot.js     ← AI Chatbot (NanangBot)
├── images/            ← Gambar & aset
├── package.json       ← NPM config (dev server)
└── README.md
```

## 🌐 Deploy ke Cloudflare Pages

1. Push ke GitHub
2. Login [Cloudflare Dashboard](https://dash.cloudflare.com/) → Pages
3. Connect GitHub repo **My-Portfolio**
4. Build settings:
   - **Framework preset:** None
   - **Build command:** *(kosongkan)*
   - **Build output directory:** `/` atau `.`
5. Deploy! ✅

## 🛠️ Teknologi

- **HTML5 + Tailwind CSS** (CDN)
- **Font Awesome 6.4.0**
- **Google Fonts** (Plus Jakarta Sans, Space Grotesk)
- **Formsubmit.co** (Contact form, gratis)
